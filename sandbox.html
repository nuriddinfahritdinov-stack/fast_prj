Подключение к песочнице...

⚡ Быстрые команды

# PostgreSQL

pg → подключиться к PostgreSQL (superuser)

sudo systemctl status postgresql → проверить статус

psql -U student drelab

 

# MongoDB

mongo → подключиться к MongoDB

sudo systemctl status mongod → проверить статус

mongosh

 

# Администрирование (Full Root Access)

sudo apt install <pkg> → установить любой пакет

sudo journalctl -u postgresql → логи сервиса

🔬 Что доступно в песочнице

👑

Полный Root доступ (sudo)

Устанавливайте пакеты (apt install), управляйте сервисами (systemctl), читайте логи (journalctl)
